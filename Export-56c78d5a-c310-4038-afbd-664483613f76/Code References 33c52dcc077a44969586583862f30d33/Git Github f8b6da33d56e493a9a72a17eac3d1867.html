Git environment Setup

Git

  • Mac installation
    • Step 1. Install Homebrew.
      $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
    • Step 2. Install Git.
      $ brew install git

SSH key

  • Step 1. Enter ssh-keygen -o -t rsa -b 4096 -C "your email" in your Git bash, press enter untill the end.
  • Step 2. Find your key at the path the bash report, for me, it’s  '/c/Users/<user_name>/.ssh/id_rsa.pub’
  • Step 3. Open the id_rsa.pub file, copy the ssh key inside to your GitLab=>Setting=>SSH keys
  • Step 4. Press Add keys, and now you can communicate GitLab with your PC now.

因為我找不到ssh-key的路徑檔案，所以我重新生成key：

$ ssh-keygen

(press `return` till the end)

(press `y` to overwrite)

然後再：

$ cat ~/.ssh/id_rsa.pub

output出來的結果那一整段就都是 SSH-key

Check all your SSH keys

$ ls -al ~/.ssh

Git pull and push

Clone - To check out a project from gitlab

  • Step 1. In your project, find the clone option and copy the link in Clone with HTTPS.
  • Step 2. In git bash, enter
$ git clone <link you copy at step2> <where you want to put the project in your PC>
- example -
$ git clone https://playlab.computing.ncku.edu.tw:4001/sckao/hw_n2609xxxx.git D:\hw_n2609xxxx
  • Step 3. Now you can see files in the path you select.

Push - update file to your GitLab

  • If you change something in project folder, you can push the changes back to the GitLab.
  • Step 1. Open git bash in your project folder
  • Step 2. enter
$ git add .
$ git commit -m "describe changes"
$ git push

now you should see changes on GitLab
