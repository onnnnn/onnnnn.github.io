Markdown is a very simple and extremely easy to learn language. It allows us to quickly write a page or blog post and then convert it to HTML.

Here’s some example markdown:

# This is a heading

## Here's a smaller heading

This is a paragraph

* Here's a bulleted list
* Another item

1. And an ordered list
1. The numbers don't matter

> This is a qoute

[This is a link to Google](https://google.com)

Let’s get started.

  1. Install Node.js.
  1. Open your terminal and install gh-markdown-cli by typing sudo npm install gh-markdown-cli -g (needed node.js 0.6.0+, 2021/07/31)
  1. Make a new folder by typing mkdir example and enter it by typing cd example.
  1. Now open your favorite text editor, nodepad is fine, and save a file called index.md in the example folder.
  1. Type # Hello, world! into index.md and save it.
  1. Return to the terminal and run mdown --input "**/*.md" --output html
  1. Inside the example folder, there is now a folder called html. Open html/index.html in your web browser to see your web page.

Excellent
